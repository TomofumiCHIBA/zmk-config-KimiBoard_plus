## 1.デフォルトファームウェアについて 
>[!NOTE]
>**・Windows・Mac上の言語設定の1と2を直接指定し、日本語・英語入力の切り替えを行う形でキーを割り当てています。**

### 1-1.キーマップ  

####  デフォルトのキーマップ  

| キー | 割り当て |
|:-|:-|
| 0キー| Bluetooth接続先変更（1→2→3のトグル） |
| 1キー| バックスペースキー|
| 2キー| スペースキー |
| 3キー| エンターキー |
| 4キー| ホイールクリック |
| 5キー| 左クリック |
| 6キー| 右クリック |
| 1キー + 4キー| 英数入力（言語設定2） |
| 3キー + 6キー| かな入力（言語設定1） |
| 4キー + 5キー| スクロールレイヤー |
| 1キー + 3キー| 上段（123キー）と下段（456キー）のキーを入れ替え |
| 1キー + 2キー + 4キー + 5キー| Bluetooth接続先変更（1→2→3のトグル） |
| 2キー + 3キー + 5キー + 6キー| 現在のBluetooth接続先情報を削除 |
